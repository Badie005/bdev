# Angular App Template

Template B.DEV pour créer des applications Angular modernes.

## Stack

- Angular 17+ (Standalone Components)
- TypeScript
- SCSS
- Angular Material (optionnel)

## Usage

```bash
bdev new angular-app mon-app
cd mon-app
npm install
ng serve
```

## Structure

```
├── src/
│   ├── app/
│   │   ├── components/
│   │   ├── pages/
│   │   ├── services/
│   │   └── app.component.ts
│   ├── assets/
│   └── styles/
└── angular.json
```

## Commandes

```bash
ng serve         # Serveur de développement
ng build         # Build production
ng generate      # Générer composants
ng test          # Tests unitaires
```
